SISTEMA DE CADASTRO DE USUÁRIOS

Um sistema simples de gerenciamento de usuário desenvolvido em Python, com operações completas de CRUD (Create, Read, Update, Delete) e persistência de dados em arquivo JSON.




FUNCIONALIDADES

* Cadastrar usuário
* Listar usuários
* Atualizar dados do usuário
* Deletar usuário
* Buscar usuário por ID



TECNOLOGIAS UTILIZADAS

* Python 3
* JSON (para armazenamento de dados)



COMO EXECUTAR O PROJETO
1. Clone o repositório:

git clone https://github.com/Leonardosbor/Sistema-de-Cadastro-de-Usu-rio-com-CRUD

2. Acesse a pasta do projeto:

cd Sistema-de-Cadastro-de-Usu-rio-com-CRUD

3. Execute o arquivo principal:

python main.py



Estrutura do projeto
|
├── main.py     # Menu e controle do sistema 
├── usuarios.py # Lógica das operações (CRUD) ├── storage.py    # Leitura e escrita no JSON 
├── usuarios.json # Banco de dados simples



OBSERVAÇÕES

* Os dados são armazenados localmente em um arquivo JSON
* O sistema roda diretamente no terminal
* Ideal para fins de estudo e prática de lógica e organização do código
